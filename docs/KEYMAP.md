# Keymap

**English** · [Português](KEYMAP.pt-BR.md)

ytmtui is designed to stay under your fingers. Most actions work with familiar
terminal movement keys and single-letter commands.

## Navigation

| Key | Action |
|---|---|
| `↑` / `↓` or `k` / `j` | Move selection up/down |
| `←` / `→` or `h` / `l` | Switch between sidebar and list |
| `Tab` | Toggle focus |
| `Enter` | Play a song, open an artist, load an album, or load a playlist |
| `1`…`9`, `0` | Jump straight to a section |
| `?` | Open help |
| `q` or `Ctrl+C` | Quit |

## Search

| Key | Action |
|---|---|
| `/` | Open search |
| typed query + `Enter` | Search songs, artists, albums, and playlists |
| `Esc` | Cancel search input |

## Playback

| Key | Action |
|---|---|
| `Space` | Play/pause |
| `n` | Next track |
| `p` | Previous track |
| `[` / `]` | Seek backward/forward 5 seconds |
| `s` | Stop |
| `w` | Open Now Playing (large cover, progress, spectrum, lyric line) |
| `+` / `=` | Volume up |
| `-` / `_` | Volume down |

## Queue and Modes

| Key | Action |
|---|---|
| `a` | Add selected track to queue |
| `z` | Toggle shuffle |
| `r` | Cycle repeat: off, all, one |

## Account

| Key | Action |
|---|---|
| `g` | Sign in or refresh cookies from browser |
| `↑` / `↓` or `k` / `j` | Select an account in the confirmation dialog |
| `Enter` | Confirm the selected browser account |
| `Esc` | Cancel account confirmation and preserve the current account |
| `f` | Like/unlike current track when signed in |

## Appearance

| Key | Action |
|---|---|
| `,` | Open Settings |
| `↑` / `↓` | Pick a setting (in Settings) |
| `←` / `→` | Change the selected setting (applies immediately, saved automatically) |
| `t` | Cycle color theme and save it |

## Lyrics

| Key | Action |
|---|---|
| playback position | Advances synced lyrics automatically |
| `↑` / `↓` or `k` / `j` | Browse lines; pauses auto-follow for 4 seconds |
| `Enter` | Jump playback to the line under the cursor |
| `Home` | Resume following the song |
| `<` / `>` | Nudge lyric timing by 0.25s (saved across sessions) |
