# Mapa de Teclas

**Português** · [English](KEYMAP.md)

O ytmtui foi desenhado para ficar debaixo dos dedos. A maioria das ações usa
movimentos familiares de terminal e comandos de uma tecla.

## Navegação

| Tecla | Ação |
|---|---|
| `↑` / `↓` ou `k` / `j` | Move a seleção para cima/baixo |
| `←` / `→` ou `h` / `l` | Alterna entre barra lateral e lista |
| `Tab` | Alterna foco |
| `Enter` | Toca uma música, abre artista, carrega álbum ou carrega playlist |
| `1`…`9`, `0` | Vai direto para uma seção |
| `?` | Abre ajuda |
| `q` ou `Ctrl+C` | Sai |

## Busca

| Tecla | Ação |
|---|---|
| `/` | Abre busca |
| texto + `Enter` | Busca músicas, artistas, álbuns e playlists |
| `Esc` | Cancela a busca |

## Reprodução

| Tecla | Ação |
|---|---|
| `Espaço` | Play/pause |
| `n` | Próxima faixa |
| `p` | Faixa anterior |
| `[` / `]` | Retrocede/avança 5 segundos |
| `s` | Para |
| `w` | Abre Now Playing (capa grande, progresso, espectro, linha da letra) |
| `+` / `=` | Aumenta volume |
| `-` / `_` | Diminui volume |

## Fila e Modos

| Tecla | Ação |
|---|---|
| `a` | Adiciona a faixa selecionada à fila |
| `z` | Alterna shuffle |
| `r` | Alterna repeat: off, todos, uma |

## Conta

| Tecla | Ação |
|---|---|
| `g` | Entrar ou renovar cookies pelo navegador |
| `↑` / `↓` ou `k` / `j` | Seleciona uma conta no diálogo de confirmação |
| `Enter` | Confirma a conta selecionada do navegador |
| `Esc` | Cancela a confirmação e preserva a conta atual |
| `f` | Curtir/descurtir faixa atual quando conectado |

## Aparência

| Tecla | Ação |
|---|---|
| `,` | Abre os Ajustes |
| `↑` / `↓` | Escolhe um ajuste (dentro dos Ajustes) |
| `←` / `→` | Muda o ajuste selecionado (aplica na hora, salvo automaticamente) |
| `t` | Troca o tema de cores e salva |

## Letras

| Tecla | Ação |
|---|---|
| posição da música | Avança letras sincronizadas automaticamente |
| `↑` / `↓` ou `k` / `j` | Navega pelas linhas; pausa o auto-follow por 4 segundos |
| `Enter` | Pula a reprodução para a linha sob o cursor |
| `Home` | Volta a acompanhar a música |
| `<` / `>` | Ajusta a sincronia da letra em 0,25s (salvo entre sessões) |
